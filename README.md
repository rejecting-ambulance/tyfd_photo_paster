## 使用說明
> 📸 **使用說明**：[開啟網頁](https://classy-kashata-988f2a.netlify.app)
>
> - 一次上傳所有照片，**重選會清空所有**。
> - 使用 ⬆️⬇️ 調整圖片順序，❌ 可刪除不需要的圖片。
> - 說明欄可填文字，按 **Tab** 跳到下一個。
> - 完成後，請 **開心地下載** 😄！
