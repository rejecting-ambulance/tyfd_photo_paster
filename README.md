# 照片黏貼表  
整理照片還要排版，怎麼那麼麻煩，來這讓你    
上傳照片打說明，下載檔案好心情  
[https://classy-kashata-988f2a.netlify.app/]

改標題，其他單位也可以用喔

## 使用說明
1. 進入網頁
2. 「一次」上傳所有照片，如果重選會全部重來
3. 箭頭可調整上下次序，[x]可以刪掉不要的照片
4. 輸入說明欄，按[tab]可以跳到下個說明欄
5. 最後當然是開心的下載囉
